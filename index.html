⚡ Debounce input lag in Contact Manager

Problem: Typing in search caused jank at >200 contacts.

Solution: Added 150ms debounce and key‑indexed rendering.

// JavaScript
const debounce = (fn, ms) => { let t; return (...a)=>{ clearTimeout(t); t=setTimeout(()=>fn(...a), ms); } };
const onSearch = debounce((q)=> filterContacts(q), 150);
input.addEventListener('input', e => onSearch(e.target.value));
