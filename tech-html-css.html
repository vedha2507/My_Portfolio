Highlights: semantic markup, CSS grid/flex, prefers-reduced-motion, dark mode.

Interesting Feature

Adaptive grid that flips columns at custom breakpoints.

/* Grid utility */
.grid-2 { display:grid; grid-template-columns: 1.2fr 0.8fr; gap:18px; }
@media (max-width: 960px){ .grid-2 { grid-template-columns: 1fr; } }

Decisions

  • Use system fonts for performance.
  • CSS variables for theme control.
  • Reduced motion support by default.
