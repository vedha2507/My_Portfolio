Interesting Feature

Search with pagination and specification pattern for flexible queries.

// Spring Data JPA Specification (snippet)
public Specification<Contact> hasQuery(String q) {
  return (r, cq, cb) -> {
    if (q == null || q.isBlank()) return cb.conjunction();
    String like = "%" + q.toLowerCase() + "%";
    return cb.or(
      cb.like(cb.lower(r.get("name")), like),
      cb.like(cb.lower(r.get("email")), like)
    );
  };
}

Decisions

  • Thin controllers, business logic in services.
  • DTOs for IO boundaries, Bean Validation for safety.
  • Repository specs to keep queries composable.
