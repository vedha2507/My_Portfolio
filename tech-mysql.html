MySQL • Structured Data & Indexing

Highlights: normalized schema, covering indexes, EXPLAIN plan checks.

Schema Snippet

CREATE TABLE contacts (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(120) NOT NULL,
  email VARCHAR(180) UNIQUE,
  phone VARCHAR(30),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_name (name)
);

Decisions

  • Unique email to prevent duplicates.
  • Index on name to accelerate search.
  • Audit fields for traceability.
