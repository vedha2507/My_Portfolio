Interesting Feature

Global exception handler returning consistent API errors.

@RestControllerAdvice
class ApiErrors {
  @ExceptionHandler(MethodArgumentNotValidException.class)
  ResponseEntity<Map<String,Object>> badReq() {
    return ResponseEntity.badRequest().body(Map.of("error","validation_failed"));
  }
}

Decisions

  • Use profiles for dev/test/prod parity.
  • Actuator for health/metrics endpoints.
  • Map exceptions to stable error schema.
